# 🌐 Responsive Navbar - Aiko78

## 📝 Description

Ce projet consiste en une barre de navigation (**Navbar**) moderne, élégante et entièrement **responsive**. Elle s'adapte automatiquement aux écrans mobiles via un menu "burger" interactif.

Le design utilise des effets de flou d'arrière-plan (**backdrop-filter**), des polices Google Fonts et des icônes FontAwesome pour une expérience utilisateur fluide et esthétique.

-----

## 🚀 Fonctionnalités

  * 📱 **Design Adaptatif** : Bascule automatiquement vers un menu mobile pour les écrans inférieurs à 990px.
  * 🍔 **Menu Burger Interactif** : Animation fluide à l'ouverture et changement d'icône dynamique (barres vers croix).
  * 🎨 **Interface Moderne** : Utilisation de **Flexbox** pour l'alignement et de la transparence avec flou pour le menu mobile.
  * ✨ **Boutons d'Action** : Intégration de boutons stylisés avec effets de survol (*hover*).

-----

## 🛠️ Prérequis

Aucune installation de dépendances locales n'est requise. Le projet utilise des **CDNs** pour :

  * 🚩 **FontAwesome 6.7.2** (icônes).
  * 🔤 **Google Fonts** (police Poppins).

-----

## ⚙️ Installation et Utilisation

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/votre-utilisateur/navbar-responsive.git
    ```
2.  **Lancer le projet** :
    Ouvrez le fichier `index.html` dans n'importe quel navigateur web. 🌍

-----

## 📂 Structure du Projet

  * 📄 `index.html` : Structure de la page et logique JavaScript pour la manipulation des classes CSS.
  * 🎨 `style.css` : Styles globaux, mise en page Flexbox et **Media Queries** pour la partie responsive.
